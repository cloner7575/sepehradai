# راهنمای کاربری (فارسی)

## دایرکت هوشمند اینستاگرام

پیام‌ها و کامنت‌های تکراری را خودکار پاسخ بده و مخاطب را به مسیر منظم سفارش هدایت کن.

1. از منوی اینستاگرام → **اتصال** → «اتصال امن اینستاگرام»
2. حساب حرفه‌ای را انتخاب کنید.
3. در Meta، Webhook را روی دامنه خود تنظیم کنید.
4. **پاسخ‌های خودکار** و **فلو** بسازید؛ برای نمونه از seed استفاده کنید ولی در production فعال نکنید تا تست شود.
5. **صندوق پیام‌ها** را برای پاسخ انسانی و توقف اتوماسیون باز کنید.

امنیت: اتصال از مسیر رسمی Meta است و رمز اینستاگرام از شما دریافت نمی‌شود.
