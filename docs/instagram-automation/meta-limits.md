# محدودیت‌های رسمی Meta

- پنجرهٔ ۲۴ساعته پیام (و قوانین فعلی Meta را رعایت کنید)
- Private Reply: معمولاً یک‌بار به ازای کامنت در بازه مجاز
- Rate limit Graph API — client با Retry-After و backoff
- بدون scraping / unofficial API / Selenium
- تشخیص فالو از API رسمی برای اتوماسیون قابل اتکا نیست → در محصول غیرفعال
- Story mention/reply فقط با webhook رسمی
