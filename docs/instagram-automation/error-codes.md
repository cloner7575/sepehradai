# کدهای خطای داخلی

پیشوند `META_*` از `map_meta_error`:

- `META_AUTHENTICATION_*` — اتصال مجدد
- `META_PERMISSION_*` — App Review / scopes
- `META_RATE_LIMIT_*` — backoff
- `META_VALIDATION_*` — ورودی نامعتبر
- `META_UNSUPPORTED_FEATURE_*` — قابلیت تأییدنشده
- `META_EXPIRED_INTERACTION_WINDOW_*` — پنجره پیام
- `META_NO_TOKEN` — توکن موجود نیست

پیام خام Meta به کاربر نشان داده نمی‌شود.
