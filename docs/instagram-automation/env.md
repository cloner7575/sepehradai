# Env

```
META_APP_ID=
META_APP_SECRET=
META_WEBHOOK_VERIFY_TOKEN=
META_REDIRECT_URI=https://yourdomain.com/instagram/oauth/callback/
META_GRAPH_API_VERSION=v21.0
META_TOKEN_ENCRYPTION_KEY=
INSTAGRAM_LINK_ALLOWLIST=yourdomain.com
```

`META_TOKEN_ENCRYPTION_KEY` می‌تواند کلید Fernet یا هر رشتهٔ قوی باشد (SHA256→Fernet مشتق می‌شود).
